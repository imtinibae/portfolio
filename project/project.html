ghibli chrome extension

L'extension vous permet d'obtenir un espace avec les informations clés de la journée, et démarrer votre navigation avec des outils pratiques! A cela s'ajoute, des informations clés sur les films pour mieux les découvrir. Elles sont récupérées grâce à une API qui regroupe des données concernant chaque film (https://ghibliapi.vercel.app/).

le comptoir

Site de vente de meuble vintage. Projet de groupe de 8 personnes sur 2 semaines, équipe front-end la première semaine, puis back-end la seconde. Projet réalisé avec le framework Chakra en CSS pour le front, et Express, node.js ainsi que Supabase pour la partie back-end.

weareclothed

Projet titre RNCP VI - En cours - Application -
Réseau social où les utilisateurs partagent leurs tenues du jour tout en spécifiant (à l'aide d'un tag) d'où proviennent les articles. Le projet a d'abord été réalisé grâce au langage Dart et son framework Flutter pour une application multiplateforme. Le projet est en cours de migration vers le langage Swift.
