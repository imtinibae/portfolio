ghibli chrome extension

L'extension vous permet d'obtenir un espace avec les informations clés de la journée, et démarrer votre navigation avec des outils pratiques! A cela s'ajoute, des informations clés sur les films pour mieux les découvrir. Elles sont récupérées grâce à une API qui regroupe des données concernant chaque film (https://ghibliapi.vercel.app/).
